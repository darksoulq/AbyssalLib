Custom Events

AbyssalLib comes with a few custom events, some are cancellable.

Name

When it is fired

BlockBrokenEvent

Fired when a CustomBlock is broken.

BlockPlacedEvent

Fired when a CustomBlock is placed.

BlockInteractionEvent

Fired when a CustomBlock is interacted with (right or left click).

EntityDeathEvent

Fired when a custom Entity dies.

EntityLoadEvent

Fired when a custom Entity is loaded into the world.

EntitySpawnEvent

Fired when a custom Entity is spawned into the world.

PacketSendEvent

Fired when a packet is sent by the server.

PacketReceiveEvent

Fired when a packet is received by the server.

RegistryApplyEvent

Fired when a DeferredRegistry is applied.

ResourcePackGenerateEvent

Fired when a resource pack is generated (ResourcePack#compile, which is auto called by ResourcePack#register).
